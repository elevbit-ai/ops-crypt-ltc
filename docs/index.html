Features

Built for real Litecoin — not a multi-chain gimmick

🔑

Real BIP39 wallets

Generate a 12-word seed or import an existing phrase / WIF. Keys stay on your device.

☁

Encrypted backup

Download or upload a .opsjson vault protected with AES-256-GCM + PBKDF2 (PIN).

📡

Live Litecoin APIs

Balance, UTXOs, history and broadcast via litecoinspace, BlockCypher and Blockchair fallbacks.

📱

Android APK ready

Capacitor project included so you can build a signed APK for phones.

How to use

Web app or APK — same wallet flow

1. Create or import

  1. Set a 6-digit PIN (encrypts the seed on device).
  2. Create a new 12-word wallet, or import BIP39 / WIF / backup file.
  3. Write the seed offline. Never share it.

2. Receive & send

  1. Receive: show QR for SegWit (ltc1…) or Legacy (L…).
  2. Send: paste LTC address + amount → signed locally → broadcast.
  3. Use small amounts first until you are comfortable.

Como usar

App web ou APK — o mesmo fluxo

1. Criar ou importar

  1. Defina um PIN de 6 dígitos (criptografa a seed no aparelho).
  2. Crie uma carteira de 12 palavras ou importe BIP39 / WIF / backup.
  3. Anote a seed offline. Nunca compartilhe.

2. Receber e enviar

  1. Receber: QR SegWit (ltc1…) ou Legacy (L…).
  2. Enviar: endereço + valor → assina localmente → transmite.
  3. Comece com valores pequenos até dominar o fluxo.
