Abrir app

MetaMask não suporta Litecoin (só Ethereum/EVM). Use a seed BIP39 na Trust Wallet, Exodus, etc., e ative Litecoin. Endereço Legacy (BIP44 m/44'/2'/0'/0/0) costuma ser o da Trust.

Recursos v2

  • Criar carteira real com 12 palavras BIP39
  • Importar seed ou WIF
  • Upload/download backup criptografado .opsjson
  • PIN + AES-256-GCM + PBKDF2 (250k)
  • APIs: litecoinspace.org, BlockCypher, Blockchair
  • Endereços SegWit (ltc1) + Legacy (L…)

Usar o app

  1. Defina PIN de 6 dígitos
  2. Criar carteira ou importar seed/WIF/backup
  3. Anote as 12 palavras offline
  4. Receba LTC (QR) · Envie com assinatura local
  5. Backup → baixar .opsjson

Gerar APK

cd C:\Users\franc\ops-crypt-ltc
npm install
npm run build
npx cap add android
npx cap sync android
npx cap open android
# Android Studio → Build → Build APK(s)

Desenvolvimento

npm run dev
